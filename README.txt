----PROYECTO COMPRA DE PRODUCTOS DE BELLEZA CON BASE DE DATOS----

Este proyecto esta desarrollado en NetBeans que permite a los usuarios seleccionar productos de belleza,
ingresar la cantidad deseada y calcular el costo total de la compra.
Cada producto tiene un precio predefinido y el sistema muestra el total a pagar.

----PROGRAMAS UTILIZADOS----

Nethbeans, SQl, Java

------FUNCIONALIDADES-----

Lista con precios fijos de los productos
El usuario ingresa la cantidad de productos deseados y este se multiplica por el precio de estos
Se pueden modificar la cantidad de productos
Eliminación de productos, total de productos y el promedio final 
Apartado para cerrar la aplicacion
 
----INSTALACIÓN Y EJECUCIÓN-----

Abrir el proyecto en NetBeans.
Configurar la base de datos SQL y cargar las tablas necesarias.
Compilar y ejecutar el proyecto desde el IDE.
No funciona correctamente al correr el proyecto en otro IDE diferente a java y sql ya que podria presentar fallas
